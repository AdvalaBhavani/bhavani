Html

HTML stands for HyperText Markup Language, and it's the backbone of nearly every webpage you see online. It’s not a programming language but a markup language—meaning it structures content rather than performs logic.

Css

CSS, or Cascading Style Sheets, is used to style and layout web pages. It allows developers to apply styles to HTML elements, such as colors, fonts, spacing, and positioning, making the web visually appealing.

JavaScript

JavaScript is a programming language that adds interactivity to web pages. It allows developers to create dynamic content, control multimedia, animate images, and much more. JavaScript runs in the browser, enabling real-time updates without needing to reload the page.

ReactJS

ReactJS is a JavaScript library used to build fast, interactive user interfaces—especially for single-page applications. It was developed by Facebook and is now maintained by a large community of developers.

AngularJS

AngularJS is a structural framework for dynamic web applications. It allows developers to use HTML as a template language and extend HTML's syntax to express application components clearly and succinctly.
